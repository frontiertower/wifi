## 4. Application Page: "Personnel Data Input Terminal"

This page will have a clean, functional design, focusing on clear data input, much like a control panel or a terminal interface.

**Aesthetics:** Dominated by the screen color (green/blue/amber), with clear input fields bordered in the same color. Minimal background noise (perhaps just the blueprint grid).

* **Header Bar:** Same dark metallic/Bakelite texture with status indicators.
* **Title:** Large, centered, and glowing: "**INITIATE PERSONNEL DATA INPUT PROTOCOL**"
